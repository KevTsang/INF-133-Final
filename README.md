Outdoor Activity App with Weather Forecast - INF 133 Project

Features:
  - Add Event
  - Edit Event
  - Mark Event Complete
  - Delete Event
  - Get weather forecast for each event

To run the project in your local machine:
  - Download the following 3 files and put them in one folder:
      - index.html
      - index.js
      - style.css
  - Run index.html
